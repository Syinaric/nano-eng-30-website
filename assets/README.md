# Assets Folder

## Images

This folder contains the website images:

- `qnc.jpg` - QNC building background image
- `ne logo.png` - Nanotechnology Engineering logo

### Recommended Image Specifications:
- **Format**: JPG, PNG, or WebP
- **Resolution**: 1920x1080 or higher
- **File size**: Under 2MB for optimal loading
- **Aspect ratio**: 16:9 or similar landscape orientation

### How to Add Your Image:
1. Save your University of Waterloo building photo as `uw-building.jpg`
2. Place it in this `assets/` folder
3. The website will automatically use it as the background

### Alternative Filename:
If you prefer a different filename, update the CSS file (`styles.css`) line 25:
```css
background-image: url('assets/your-filename.jpg');
```

### Image Sources:
- Take a photo of the University of Waterloo campus
- Use an official UW building photo
- Find a high-quality image of the engineering buildings
